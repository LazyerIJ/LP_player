# v1.0.1

**Released: 2026-03-30**

## New Features
- Tracklist panel: browse and switch songs from the LP player (☰ button)
- Queue extraction from YouTube Music DOM
- Fullscreen toggle button (⛶ bottom-right)
- Wake Lock: prevent screen sleep during playback
- Ko-fi support link in popup

## Bug Fixes
- Fix tonearm direction: rest position off record, active position on record
- Fix tonearm clamp and drag threshold logic
- Restore tonearm active class after drag ends during playback
- Reset progress bar on song change (prevent stale time data)

## Improvements
- Slow record spin speed (1.8s → 3s) for more realistic feel
- Album art: less cropping with contain + scale(1.35)
- Background service worker relays GET_QUEUE and PLAY_TRACK messages
