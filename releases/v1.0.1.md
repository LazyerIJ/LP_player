# v1.0.1

**Released: 2026-03-30**

## New Features
- Wake Lock: prevent screen sleep during playback

## Bug Fixes
- Reset progress bar on song change (prevent stale time data)
- Fix record spin stuttering: separate box-shadow into static layer, GPU-accelerate rotation
- Fix progress bar causing layout reflow: use scaleX transform instead of width
- Fix tonearm click/drag confusion: add 6px drag threshold to distinguish click from drag
- Fix tonearm click misfire after drag end
- Fix stale progress flash on song change (force reset and early return)

## Improvements
- Realistic tonearm playback sequence: record spins up → tonearm moves → music plays
- Skip redundant DOM updates with dirty checking (playback state, song info, progress)
- Cache ambient background gradient to avoid recomputation
- Skip tracklist re-render when queue is unchanged
